# Product Requirements

Create a browser playable voxel world with procedural terrain around the player, Three.js rendering, a HUD, an inventory bar, movement controls, a debug state surface, and domain shaped state for movement, inventory, world updates, and sequence objectives.

The public game should expose window.GameHost.getState() with player, world, inventory, domainTrace, events, input, sequence, and renderer diagnostics.
