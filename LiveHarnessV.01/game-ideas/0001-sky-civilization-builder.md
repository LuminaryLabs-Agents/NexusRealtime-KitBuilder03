# Idea: Sky Civilization Builder

Make 10 voxel games about building connected sky islands, airships, bridges, wind currents, resource towers, and floating settlements.

Must include:
- procedural floating islands or high-altitude terrain
- RPC build commands
- inventory/tool selection
- world loading around the player
- GameHost state
- kit-aware integration diagnostics

Each variant should feel like a separate prototype, not a reskin.
