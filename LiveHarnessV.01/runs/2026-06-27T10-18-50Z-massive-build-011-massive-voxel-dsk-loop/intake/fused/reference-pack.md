# Fused Repo Capability Pack

- nexusrealtime.domain-service-kit: Domain Service Kit contract should constrain generated domains.
- nexusrealtime.sequence-node: SequenceNode is a useful AST pattern for objectives and kit deployment.
- nexusrealtime.procedural-world: Procedural and terrain systems should produce descriptors consumed by host/renderers.
- protokit.world-terrain-pattern: World/terrain ProtoKits are patterns for chunk and patch lifecycle.
- protokit.render-descriptor-pattern: Visual pipeline kits output render descriptors rather than owning a renderer.
- protokit.action-input-kit: Action input kit pattern routes host input into semantic game actions.
